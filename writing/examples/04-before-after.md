# Worked Example: Before and After (Transformation Reveal)

> Format reference: `references/storytelling.md` § 4.
> Mediums: short-form video script (~15s) + LinkedIn-post translation.
> Beats hit: Hook → Before State → Transition → After State → Context → CTA.

---

## Short-form video script (15 seconds)

**[HOOK / 0–2s]**
Spoken: "6 months. 30 seconds."
On-screen text: "6 months → 30 seconds"

**[BEFORE STATE / 2–6s]**
No words. Camera pans across the kitchen.
Text overlay: "218 lbs. Couldn't take stairs without stopping."

**[TRANSITION / 6–7s]**
Hard cut on a beat drop. Same kitchen, same shirt, same lighting.

**[AFTER STATE / 7–11s]**
Text overlay: "168 lbs."

**[CONTEXT / 11–14s]**
Spoken: "No surgery. No Ozempic. Just protein, walking, and one Sunday meal-prep."

**[CTA / 14–15s]**
Text overlay: "Want the macros? Comment PLAN."

---

## LinkedIn-post translation (~80 words)

6 months. Same kitchen. Same shirt. Same lighting.

[Photo 1: 218 lbs] [Photo 2: 168 lbs]

No surgery. No Ozempic. Just protein, walking, and one Sunday meal-prep.

Want the exact macros? Comment PLAN.

---

**Why this works:** the gap is large and visual, the transition is timed to a beat (the source's "emotional peak"), and "no surgery / no Ozempic" pre-empts the disqualifying objection. The format works because it doesn't try to explain — the visual gap does the work.
